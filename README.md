# Echo bot (no frameworks)
Test project for Metalamp. Description of requirements aviable [here](https://coda.io/@metalamp/education/3-14) and [here](https://coda.io/@metalamp/education/5-16).

## Architecture
Bot relies heavily on MTL-pattern. At current moment it can work with console, vkontakte and telegram. All it main logic are located in bot folder and represented throught IsFrontEnd and FrontEndIO classes.
